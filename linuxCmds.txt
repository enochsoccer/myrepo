/----LINUX COMMANDS----/



/----APPLICATIONS----/
Install snap | sudo snap install <appName>
Search snap (aka application) in order to download | snap find <appName>
Searching directory location of downloaded app | whereis <appName>

/----FILE MANAGER----/
Create folder in current directory | mkdir <folderName>
Open file on its application | xdg-open <filenamehere.ext>
Open folder with file manager | nautilus <foldername>
Rename file in same directory | mv <currentname> <newname>


/----TEXT EDITOR----/
Go back to accidentally quit nano | fg
Open .txt file with nano | nano <filenamehere.txt>
